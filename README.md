# collectivedynamics.org

Website of the International Association for Collective Dynamics (IACD) e.V.

Built with [Hugo](https://gohugo.io/) and the [Hextra](https://github.com/imfing/hextra) theme.
Deployed to GitHub Pages via GitHub Actions on push to `main`. Custom domain: `collectivedynamics.org` (see `CNAME`).

## Local development

```
hugo mod get -u
hugo server
```
